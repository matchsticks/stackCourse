CSS selectors

To inherit style attributes we can use CSS selectors in order to pass some specific styles relying on relation, meaning that for example we can set the style for childrens of a determined tag by using one of these selectors. On the following table we have these selectors and its designation.

a,b Multiple element selector
a b Descendant selector
a > b Child selector
a + b Adjacent sibling selector
[a=b] Attribute selector
a:b Pseudo class selector
a::b Pseudoelement selector
